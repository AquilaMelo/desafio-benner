Gerador de Arquivo TXT em C#

Aplicação console em C# responsável por gerar arquivos TXT a partir de dados em JSON, seguindo leiautes e regras de totalização.

Tecnologias

C#

.NET

Newtonsoft.Json

NUnit

Como executar

Clone o repositório

Abra a solution no Visual Studio

Execute o projeto console

Informe o arquivo JSON e o diretório de saída quando solicitado
